Iris Dark GTK
====
####Description

A Minimal, Dark GTK Theme For GTK 3.12+ and GTK 2.

####Dependencies

GTK:
* Murrine GTK Engine
* Adwaita GTK Engine

Shell:
* Liberation Sans ( Free ) ([ Shell and UI Font ](http://dl.dafont.com/dl/?f=liberation_sans)) ([ Matching Terminal Font ](http://dl.dafont.com/dl/?f=liberation_mono))
